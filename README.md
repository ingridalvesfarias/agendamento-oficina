# 🚗 Agendamento de Oficina Pro

Sistema de agendamento mobile-first desenvolvido para oficinas mecânicas que buscam profissionalismo e agilidade no atendimento via WhatsApp.

## 🛠️ Tecnologias Utilizadas
* **HTML5** & **CSS3**: Estrutura e estilização com foco em *Dark Mode* e *Glassmorphism*.
* **JavaScript (ES6)**: Lógica de validação de datas (bloqueio de fins de semana) e integração com API do WhatsApp.
* **Google Fonts**: Utilização da fonte 'Inter' para uma tipografia moderna.

## ✨ Funcionalidades
* 📱 **Design Responsivo**: Interface otimizada para dispositivos móveis e desktop, centralizada via `dvh`.
* 🗓️ **Validação de Calendário**: Impede agendamentos em datas passadas e em fins de semana (atendimento de segunda a sexta).
* 💬 **Integração WhatsApp**: Gera uma mensagem estruturada com todos os dados do veículo (Placa, KM, Ano) e do cliente.
* 🌙 **Minimalismo**: Interface em modo escuro (Dark Mode) para reduzir o cansaço visual.
